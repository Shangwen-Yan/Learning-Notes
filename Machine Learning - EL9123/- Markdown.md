- 一个回车表示空格！行尾两个空格+一个回车表示换行！！

- 用\\转义特殊字符，比如.

- 要换格式必须要加空白行！！！！

- 分割横线：三个-或\*或+

- 空格和换行

 \<br/\> 或者 两个空格+回车表示换行

 回车不表示换行

 中间有空白行(相当于前后用\<p\>包住)代表新的段落（新的格式）
